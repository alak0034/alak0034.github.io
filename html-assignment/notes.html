HTML Tags and Their Descriptions

<h1> - <h6>
Defines HTML headings, where <h1> is the highest (or most important) level, and <h6> is the least.
<p>
Defines a paragraph in HTML.
<a>
Defines a hyperlink, which is used to link from one page to another.
<img>
Embeds an image in a web page.
<ul> & <li>
Defines an unordered list and list items within it.
